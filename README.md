# QuizVerse

## Overview
QuizVerse is an interactive quiz application developed as part of Task 3 of the Web Development Internship. The project demonstrates advanced HTML, CSS, and JavaScript concepts through an engaging and responsive user interface.

## Features
- Interactive Quiz with Multiple Categories
- Real-Time Score Tracking
- Progress Bar
- Programming Joke API Integration
- Responsive Design using Flexbox, CSS Grid, and Media Queries
- Restart Quiz Functionality
- Modern and User-Friendly Interface

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Official Joke API

## Learning Outcomes
This project demonstrates:
- Responsive Web Design
- DOM Manipulation
- API Integration using Fetch
- Interactive User Interface Design
- CSS Flexbox and Grid
- JavaScript Event Handling
